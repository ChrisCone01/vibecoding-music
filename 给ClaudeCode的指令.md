你好 Claude Code。请按本文档帮我（用户）在这台 **Windows 电脑**上把一个网页项目运行起来，最终目标是浏览器打开 `http://127.0.0.1:8080/` 能正常使用（含网易云音乐搜索/播放）。

请**严格按顺序执行**，每一步做完确认结果再进行下一步；遇到判断点按文档里的分支处理；卡住了就把具体报错告诉我。

## 背景（你需要理解的架构）

- 这是个纯前端网页 + 两个本地 Node 服务，**不需要**数据库/构建/打包/公网/域名。
- 媒体（封面/音频/歌词/MV）**全部实时从网易云音乐在线获取**，经本地 Node 代理转同源。
- 必须**同时运行两个本地 Node 服务**：
  - **端口 3000**：网易云接口服务（第三方包 `NeteaseCloudMusicApi`，入口 `netease/node_modules/NeteaseCloudMusicApi/app.js`，依赖已随仓库克隆下来）
  - **端口 8080**：主站 + `/ncm/*` 代理（`server.js`，纯 Node 内置模块）
- 浏览器只访问 8080，8080 内部再调 3000。**两个都得开**，只开一个会导致音频/封面/歌词/MV 全部加载失败。

## 第 1 步：检查并安装 Node.js

执行：
```
node -v
```
- **有版本号输出**（如 v18 / v20 / v24）→ 跳到第 2 步。
- **报错/无输出**（没装 Node）→ 帮我安装 Node.js LTS：
  - 优先用包管理器：`winget install OpenJS.NodeJS.LTS`
  - 不行就提示我去 https://nodejs.org 手动下载 LTS 安装，装完重开终端，再 `node -v` 确认。

---

## 第 2 步：克隆仓库

选一个目录（如 `D:\`）执行：
```
git clone https://github.com/ChrisCone01/vibecoding-music.git
cd vibecoding-music
```
- 若没有 git：`winget install Git.Git` 后重试，或让我手动装。
- 仓库**已包含 `netease/node_modules`**（依赖随仓库带来），正常情况**无需 npm install**。
- 仅当后续启动报"找不到模块"时，才执行：
  ```
  cd netease
  npm install --registry=https://registry.npmmirror.com
  cd ..
  ```

---

## 第 3 步：【最关键】先验证这台电脑/这个网络能否访问网易云

整个项目强依赖网易云在线接口。**先单独测，否则后面白忙。**

先启动网易云接口服务（开一个终端，让它持续运行）：
```
cd netease
set PORT=3000
node node_modules\NeteaseCloudMusicApi\app.js
```
看到类似 `server running @ http://localhost:3000` 说明服务起来了。

**另开一个终端**测试：
```
curl "http://127.0.0.1:3000/search?keywords=test&limit=1"
```
判断：
- **返回一段含 `songs` 的 JSON** → ✅ 网络通网易云，继续第 4 步。
- **超时 / 返回空 / 报错 / 长时间无响应** → ❌ 这台电脑的网络（很可能公司防火墙）访问不了网易云（`music.163.com`、`*.music.126.net`、`*.vod.126.net`）。**这种情况下项目无法正常使用，且任何工具都无法绕过公司防火墙。请把这个结果告诉我**，需要更换网络环境（如手机热点）才能用。不要继续往下做无用功。

---

## 第 4 步：启动整个项目

网易云接口服务（3000）保持运行，**再开一个终端**启动主站：
```
cd vibecoding-music    （回到项目根目录）
node server.js
```
看到 `Serving ... at http://127.0.0.1:8080/` 即成功。

> 也可以直接双击项目根目录的 `启动SoundDrift.bat`（它会自动起这两个服务并打开浏览器）。但如果你已经手动起了 3000，再双击会重复启动，建议二选一。

然后用浏览器打开：`http://127.0.0.1:8080/`

---

## 第 5 步：验证功能跑通

请确认：
1. 页面出现一排卡片（半圆弧排列），顶部有视频在播，整体在缓慢运动。
2. 点击屏幕**底部中间那条白色竖线** → 弹出全屏模糊 + 大字输入框。
3. 输入一个歌名（如 `周杰伦` 或 `Shoot`）按回车 → 出现 `0% → 100%` 进度 → 搜到的歌飞到屏幕中央开始播放（有封面、有声音、下方有逐字歌词、顶部有 MV）。
4. 点击中央卡片 → 返回首页，卡片并入圆弧继续运动。

全部正常 = 部署成功。

## 可能遇到的问题与处理

- **页面能打开，但点开卡片没声音/没封面/没 MV**：基本是网络访问不了网易云（回第 3 步确认）。
- **端口被占用**（8080 或 3000 启动报 `EADDRINUSE`）：说明端口被别的程序占了。可以改端口——编辑 `server.js` 顶部的 `PORT`(8080) 和 `NCM_API`(3000)，以及 `启动SoundDrift.bat` 里的 `PORT=3000`，三处保持对应后重启。
- **启动报找不到模块**：执行第 2 步末尾的 `npm install`（需要网络能访问 npm）。

1. 确保 3000 服务在运行。
2. 项目根目录执行：`node qrlogin.js` → 会生成二维码图片 `ncm_login_qr.png`。

4. 执行：`node qrcheck.js` → 自动把登录凭证写入 `ncm_cookie.txt`。
5. 重启 `server.js`（关掉重新 `node server.js`）。这 4 首即可整首播放。
